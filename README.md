# nba-data-mining
Repo for the NBA basketball data mining course

# Installation

1. Create virtual env
```
python -m venv venv --copies
```

2. Activate venv
```
source venv/bin/activate
```

3. Install dependencies
```
pip install -r requirements.txt
```

4. Run scripts
```
cd scripts/
python xgboost-team-outcome.py
```
